# Botasaurus GMB Scraper (Starter)

This starter matches the step-by-step instructions I gave you in chat.

## Files
- `gmb_scraper.py` — the scraper script
- `requirements.txt` — Python deps
- `.env.example` — copy to `.env` and fill values
- `README.md` — you are here

## Quick start
1) Install Python 3.10+ and Google Chrome
2) Create and activate a virtual env
3) `pip install -r requirements.txt`
4) Copy `.env.example` to `.env`, fill values
5) `python gmb_scraper.py`
